# 更新日志

## [1.1.0] - 2026-04-24

### 改动

- **精简卡片布局** - 去掉套餐名、已用额度、总额度，只显示剩余额度
- **智能进度条** - 以所有账号中的最高余额为 100%，全部低于 $100 时以 $100 为 100%
- **新增 `account` 字段** - 配置中每个账号新增 `account` 字段，卡片上显示登录用户名

### 新增

- **单独刷新** - 每张卡片有独立刷新按钮，无需等待全部刷新
- **响应式网格** - 卡片最小宽度 240px 自动填充，适配手机、半屏、全屏

## [1.0.0] - 2026-03-27

### 新增

- **配置化刷新间隔** - 从 `config.json` 读取 `refreshInterval`，自动匹配或添加自定义选项
- **PHP 启动说明** - 新增 `PHP启动说明.md`，本地 PHP 服务器启动指南

## [0.0.1] - 2026-03-27

### 首次发布

- 多平台 API 账号监控大屏
- 实时配额追踪与进度条展示
- 可配置自动刷新间隔
- 深色主题监控界面
